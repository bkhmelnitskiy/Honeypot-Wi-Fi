# Software

### C

- processing packets with eBPF (XDP)
    - [eBPF](https://ebpf.io/)
    - [bcc](https://github.com/iovisor/bcc)

### Go or Python

- processing packets with eBPF
    - [PythonBPF](https://github.com/pythonbpf/Python-BPF)
- connection with a mobile device via Bluetooth Low Energy
    - [bless](https://github.com/kevincar/bless)
    - [Go Bluetooth](https://github.com/tinygo-org/bluetooth)
- connection with wpa_supplicant via a unix socket or D-Bus
    - [python-sdbus](https://github.com/python-sdbus/python-sdbus)
    - [Go D-Bus](https://github.com/godbus/dbus)
- further processing results from eBPF
    - [ebpf-go](https://ebpf-go.dev/)
- processing results from network analysis tools
    - [Zeek](https://zeek.org/)
- storing results in a local cache
    - [SQLite](https://sqlite.org/)
    - [Redis](https://redis.io/)
    - [Bolt](https://github.com/boltdb/bolt)

### JavaScript

- mobile application
    - [React Native](https://reactnative.dev/)
    - [Tauri](https://v2.tauri.app/)
- connection with an IoT device via Bluetooth Low Energy
    - [tauri-plugin-blec](https://crates.io/crates/tauri-plugin-blec)
- web server
    - [NestJS](https://nestjs.com/)

### SQL

- database queries
    - [PostgreSQL](https://www.postgresql.org/)
